>#系统的结构

系统总体结构设计采用了 MVC 的设计模式，和完全使用面向对象的思想开发，并根据 ThinkPHP框架的规则去规划视图view、控制器Controller和Model类及调用关系。提高代码的易维护性、易读性，增加类内部的纯度、类之间调用的灵活性。
